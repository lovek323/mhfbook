\question{
    Which of the following is the number $78.5$?

    \begin{enumerate}
        \item Integer
        \item Rational
        \item Variable
        \item Constant
    \end{enumerate}
}{18-solved-integer-constant.html}{31 March 2005}

\paragraph{Solution.}\partSolutionAuthor{MathGuru}

The number $78.5$:

\begin{enumerate}
    \item is not an integer because it is not a whole number
    \item is a rational because it can be expressed as the ratio of two numbers
        ($\frac{157}{2}$)
    \item is not a variable because it is a constant (see next)
    \item is a constant because it is a literal number
\end{enumerate}
